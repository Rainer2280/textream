## 📖 What is textream?

textream is a free app for macOS designed to help streamers, interviewers, and presenters read their scripts smoothly. It works like a teleprompter, showing your script on screen and highlighting the words as you speak. This way, you can keep your eyes on the camera or audience without losing your place.

The app uses a stylish Dynamic Island overlay, making the text easy to see without covering your main screen. It updates in real time, so you always know exactly where you are in the script.

## 📋 How to Use textream

Running textream is straightforward, even if you haven’t used teleprompters before. Follow these steps to display your script and start your presentation:

### Step 1: Prepare Your Script

- Create a plain text file (.txt) or copy your script content.
- You can write or edit your script in any text editor like TextEdit or Notes.

### Step 2: Load the Script into textream

- Open textream.
- Click on the "Load Script" button.
- Browse your Mac to find and select your script file.
- Your script will appear in the app window with each line ready to be highlighted.

### Step 3: Adjust Settings

- Use the settings panel to control the font size, color, and speed of text highlighting.
- You can also choose where on the screen the teleprompter appears.
- If the Dynamic Island overlay is on, you will see the script floating neatly over your desktop.

### Step 4: Start Speaking and Reading

- Start your presentation or stream.
- As you speak, textream will highlight the words automatically.
- If the app does not highlight words correctly, you can manually advance the text with the spacebar or arrow keys.

### Step 5: Pause or Stop

- Press the pause button if you need to stop temporarily.
- Close the app when your presentation ends.

## 🛠 Troubleshooting

If you run into problems, try these solutions:

- **App won’t open:** Check your macOS version meets the requirements. Re-download and reinstall if needed.
- **Script not loading:** Ensure your script is saved as a plain text (.txt) file. Avoid special formatting or images.
- **Highlighting is off:** Toggle the auto-highlight function off and on. Try manual control with keyboard keys.
- **Overlay not visible:** Make sure you did not minimize or hide the app window by mistake.
- **Performance issues:** Close other heavy apps to free memory and restart textream.
